# Навигация АнтиМинус — lo-fi

## Форм-фактор

PWA, mobile-first, ширина контента ≤ 480 px.
**Тип навигации: Stack (стек экранов)** — таб-бара и сайдбара нет,
флоу линейный (мастер из 5 шагов).

## Шапка (общий паттерн)

```
Мобильный:
┌──────────────────────────────────────────┐
│ [← Назад]    Заголовок экрана    [⚙]    │
└──────────────────────────────────────────┘

Десктоп (≥ 768 px):
┌──────────────────────────────────────────┐
│ АнтиМинус                        [⚙]    │
│──────────────────────────────────────────│
│ [← Назад]    Заголовок экрана            │
└──────────────────────────────────────────┘
Контент центрируется до 480 px.
```

Прогресс мастера: строка «Шаг N из 5» под шапкой на экранах Э3–Э7.
Э1 — без шапки, Э2 — без кнопки «Назад».

## Иерархия экранов

```
Э1  Вход  ──────────────────────────────────── root, без шапки
 └──►  Э2  Список расчётов  ─────────────────── home
           │  шапка: АнтиМинус · [+ Новый расчёт] · [⚙]
           │
           ├─ [+ Новый расчёт] ──────────────────────────────────┐
           │       └──►  Э3  Новый комплекс     шаг 1/5          │
           │                   └──►  Э4  Параметры объекта       │
           │                              шаг 2/5                │
           │                         └──►  Э5  Скрытые позиции  │
           │                                    ★ АГА · шаг 3/5  │
           │                              └──►  Э6  Риск заказа │
           │                                    ★ CTA · шаг 4/5  │
           │                                └──►  Э7  Итоговая  │
           │                                         смета 5/5  ◄┘
           │
           └─ [тап по карточке расчёта] ──► Э7  Итоговая смета
                                            (открыть готовый)

    ·  Настройки  ──► отдельный экран, вход через ⚙ в шапке
    ·  Онбординг  ──► отдельный мастер, первый вход, заглушка
       (оба за пределами основного пути)
```

## Переходы

| Откуда | Действие | Куда |
|--------|----------|------|
| Э1 | Войти (успех) | → Э2 |
| Э2 | + Новый расчёт | → Э3 |
| Э2 | Тап по карточке | → Э7 |
| Э3 | Далее → | → Э4 |
| Э4 | Проверить комплекс | → Э5 |
| Э5 | Подтвердить и перейти к риску → | → Э6 |
| Э6 | Сформировать смету → | → Э7 |
| Э6 | ← Пересчитать | → Э4 |
| Любой экран | ← Назад | предыдущий |
