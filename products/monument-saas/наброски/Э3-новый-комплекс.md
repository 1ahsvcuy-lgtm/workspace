## Состояние: шаблон выбран

Выбранная карточка обведена рамкой (активное состояние).
Кнопка «Далее →» становится активной.

## Состояние: прайс не заполнен (предупреждение)

Над кнопкой «Далее →» появляется баннер:

```
│ ╔══════════════════════════════════╗ │
│ ║ Прайс не заполнен. Позиции      ║ │
│ ║ будут найдены, но суммы          ║ │
│ ║ покажем нулями.                  ║ │
│ ║ [Заполнить прайс →]              ║ │
│ ╚══════════════════════════════════╝ │
```
