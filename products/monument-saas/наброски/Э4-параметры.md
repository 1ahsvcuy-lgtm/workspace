## Полный режим (секция «Подробности» развёрнута)

После нажатия «▸ Добавить подробности» секция раскрывается:

```
│ ┌────────────────────────────────┐   │
│ │ ▾ Дополнительные параметры     │   │
│ │                                │   │
│ │ Размер участка (м²)            │   │
│ │ ┌────────────────────────────┐ │   │
│ │ │ 6                          │ │   │
│ │ └────────────────────────────┘ │   │
│ │                                │   │
│ │ Тип грунта                     │   │
│ │ [Обычный] [Глина] [Песок] [Скала]  │
│ │                                │   │
│ │ Способ монтажа                 │   │
│ │ [Стандартный]  [Ручной]        │   │
│ │                                │   │
│ │ Демонтаж старого               │   │
│ │ [Да]  [Нет]                    │   │
│ │                                │   │
│ │ Доступ техники                 │   │
│ │ [Есть]  [Нет]                  │   │
│ │                                │   │
│ │ Дальность доставки (км)        │   │
│ │ ┌────────────────────────────┐ │   │
│ │ │ 25                         │ │   │
│ │ └────────────────────────────┘ │   │
│ └────────────────────────────────┘   │
```

## Состояние: ошибка валидации

Незаполненное поле подсвечивается, под ним — подсказка:

```
│ Тип                                  │
│ ┌────────────────────────────────┐   │
│ │                              ▾ │   │  ← красная рамка
│ └────────────────────────────────┘   │
│  ! Укажите тип элемента              │
```
