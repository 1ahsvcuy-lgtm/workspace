# SaaS «Расчёт заказа на памятник» — дизайн-прототип

## Что это

Рабочая папка **Модуля М5** — этапа дизайна в методологии G·GENERATE².

Цель этапа: создать **кликабельный дизайн-прототип**, который можно показать
пользователям и командe для проверки гипотез до написания кода.

## Что здесь есть

| Папка | Содержимое |
|-------|------------|
| `context/` | Входные карты: Карта инсайта, Чертёж MVP, AI-двойник |
| `spec/` | Техническое задание и описание экранов |
| `content/` | Голос продукта, тексты для каждого экрана |
| `наброски/` | Lo-fi вайрфреймы (бумага / Excalidraw / Figma Jam) |
| `design/` | Референсы, визуальный паспорт, экспорты Stitch |
| `прототип/` | Кликабельный прototип (Figma / ссылки / экспорты) |

## Что здесь НЕТ

Кода рабочего приложения — он появится в следующем модуле, **M·MAKE**.

## Источник правды

Карты в папке `context/` + NotebookLM проекта.
