# прототип/ — кликабельный прототип

Сюда кладём финальные материалы:

- `ссылка.md` — ссылка на прототип в Figma / другом инструменте
- Экспорты экранов (PNG / PDF)
- Видео-демо прохода по сценарию (опционально)

## Критерий готовности

Прototип считается готовым, когда:
1. Все экраны из Чертежа MVP присутствуют
2. Все кликабельные переходы работают
3. Основной сценарий проходится от начала до конца без тупиков
