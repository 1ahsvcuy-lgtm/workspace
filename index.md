## Команды системы

| Команда | Действие |
|---|---|
| `обработай raw/файл.md` | Читает источник → компилирует в KB → обновляет [[index]] + [[log]] |
| `обработай raw/` | Пакетная обработка всех необработанных файлов |
| `проверь систему` | Lint-чеклист → доклад одним блоком |
| `разбери inbox` | Читает `inbox/` → предлагает куда перенести каждый файл |
| `план недели` | Приоритеты из [[Tasks Board]] + [[MISSION_CONTROL]] |
| `итоги недели` | Отчёт + анализ + обновление файлов |
| `новый проект [название]` | Создаёт `projects/папку` + README + обновляет [[index]] + [[log]] |
| `сохрани это` | Сохраняет ценный ответ в `knowledge-base/` или `projects/` |
