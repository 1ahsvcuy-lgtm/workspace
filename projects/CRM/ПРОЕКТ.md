## Суть

Сейчас бизнес работает в ручном режиме: нет воронки, нет учёта статусов заказов, нет системы уведомлений клиентов. Это главная причина, по которой клиенты жалуются на «информационный вакуум после оплаты» — провал №1 на рынке.

**Две задачи одним внедрением:**
1. CRM — учёт заказов, воронка продаж, аналитика
2. WhatsApp-трекинг — автоматические или ручные уведомления клиенту по этапам

## WhatsApp-трекинг: этапы и шаблоны сообщений

После получения предоплаты — клиент автоматически получает сообщения при смене статуса заказа.

### Шаблон 1 — Заказ принят в работу
```
Добрый день, [Имя]!

Ваш заказ №[номер] принят в производство.

Что входит в ваш заказ:
✓ [Тип памятника, материал]
✓ Кладбище: [название]
✓ Срок готовности: [дата]

Мы будем присылать вам фото по каждому этапу — ничего не нужно уточнять самостоятельно.

С уважением, мастерская «Ангел»
```

### Шаблон 2 — Камень получен и распилен
```
[Имя], обновление по вашему заказу №[номер]:

📦 Камень получен на склад и передан в обработку.
[ФОТО камня]

Следующий этап — подготовка эскиза портрета. Пришлём на согласование.
```

### Шаблон 3 — Эскиз портрета на согласовании
```
[Имя], готов эскиз портрета по заказу №[номер].

[ФОТО/ФАЙЛ эскиза]

Пожалуйста, проверьте сходство и напишите:
— «Утверждаю» — если всё верно
— Ваши правки — если нужно скорректировать

Мы не приступаем к гравировке до вашего подтверждения.
```

### Шаблон 4 — Гравировка выполнена
```
[Имя], портрет нанесён на камень ✓

[ФОТО готовой гравировки]

Заказ №[номер] переходит на этап подготовки к монтажу.
```

### Шаблон 5 — Фундамент / скрытые работы
```
[Имя], начали монтаж на кладбище [название], заказ №[номер].

Фото скрытых работ (фундамент):
[ФОТО траншеи]
[ФОТО армирования]
[ФОТО заливки бетона]

Глубина: [X] мм | Арматура: Ø[X] мм | Бетон: М[X]

Ждём набора прочности [X] дней, после — установка стелы.
```

### Шаблон 6 — Монтаж завершён
```
[Имя], ваш заказ №[номер] выполнен ✓

[ФОТО итогового вида — общий план]
[ФОТО итогового вида — крупный план]

Кладбище: [название] | Участок: [номер]

Для окончательной приёмки и оплаты остатка свяжитесь с нами или приедьте на объект.
Также прилагаем Инженерный паспорт объекта.
```

### Шаблон 7 — Запрос отзыва (через 2–3 дня после закрытия)
```
[Имя], спасибо, что доверили нам это важное дело.

Если вас всё устроило — будем очень признательны за отзыв на Яндекс.Картах или 2GIS. Это помогает другим семьям найти нас в трудный момент.

[Ссылка на Яндекс.Карты]
[Ссылка на 2GIS]

Мастерская «Ангел», Первоуральск
```

## Что нужно для запуска

### Технически
- [ ] Выбрать CRM (amoCRM / Битрикс24)
- [ ] Зарегистрировать аккаунт, настроить воронку (этапы выше)
- [ ] Подключить интеграцию с WhatsApp (Wazzup или WABA)
- [ ] Создать шаблоны сообщений в системе (7 шаблонов выше)
- [ ] Подключить номер телефона к WhatsApp Business API
- [ ] Настроить Яндекс.Метрику на сайте + передачу лидов в CRM

### Регламент для команды
- [ ] Написать инструкцию: как менеджер создаёт карточку заказа
- [ ] Написать инструкцию для мастеров: что и когда фотографировать (этапы 3–6)
- [ ] Назначить ответственного за отправку сообщений (менеджер или автоматически)
- [ ] Провести обучение команды (1–2 часа)

### Интеграция с Инженерным паспортом
- [ ] В момент закрытия заказа — менеджер заполняет паспорт и прикрепляет к карточке CRM
- [ ] Скан паспорта + фото отправляются клиенту в WhatsApp финальным сообщением
