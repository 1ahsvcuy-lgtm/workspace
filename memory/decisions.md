## [2026-06-27] Архитектура системы

**Решение:** Единый корень `C:\Users\user\Documents\WORKSPACE\` — одно место для базы знаний, разработок и агентов.

**Обоснование:** Контекст не должен теряться между сессиями. Все агенты читают один источник правды. Синхронизация через GitHub → VPS.

**Статус:** активно

---

## [2026-06-27] Структура памяти агентов

**Решение:** Агенты пишут в `memory/log.md` (хронология) и `memory/sessions/ДАТА_агент.md` (детали). Решения — в `memory/decisions.md`.

**Обоснование:** Долгосрочная память через файлы, краткосрочная — через сессионные файлы. Новый агент читает log.md + нужный продукт README.md и сразу в контексте.

**Статус:** активно
