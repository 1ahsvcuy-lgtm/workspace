# raw/ — Сырые источники

Внешние источники — статьи, PDF, ссылки через Web Clipper.
НЕ редактировать вручную — это сырые источники.
По команде `обработай` — Claude читает и компилирует инсайты в knowledge-base.
Формат файлов: `ГГГГ-ММ-ДД_название.md`

---

## Как добавить источник

1. Сохрани файл в эту папку с именем по формату: `2026-04-29_название-статьи.md`
2. Напиши Claude: `обработай raw/2026-04-29_название-статьи.md`
3. Claude прочитает источник и скомпилирует инсайты в нужные файлы `knowledge-base/`
4. Запись об операции появится в `log.md`

## Правила

- Файлы в `raw/` никогда не удаляются и не редактируются — только читаются
- Один файл = один источник
- После обработки файл остаётся в `raw/` как архив
