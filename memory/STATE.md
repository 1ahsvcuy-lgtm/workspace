## Система

- Единый Obsidian-vault и workspace: `C:\Users\user\Documents\WORKSPACE`.
- Канонические правила агентов: [[AGENTS]]. Карта файлов: [[index]].
- Долговременная память: [[memory/decisions]], [[memory/log]] и `memory/sessions/`.
- Все изменения, задачи и решения фиксируются с датой, временем и именем агента.
- GitHub `origin/main` — центральная копия; безопасная синхронизация Windows/VPS описана в `scripts/sync/README.md`.
